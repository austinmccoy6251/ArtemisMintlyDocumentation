You can create groups out of the different items that appear on your Network. This can be used to organize your work and quickly highlight various aspects of it. More importantly, groups are used to create coherent Views.

Views are a useful tool to compare Fields across different Resources. The let you create tables like this one that are most often used to display parameters or compare variables.
