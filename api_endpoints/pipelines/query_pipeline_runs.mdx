---
title: "Query Pipeline Runs"
api: "GET https://artemisarbackend.com:8080/ext/query_pipeline_runs"
description: "This endpoint queries and returns a specified run of a specified pipeline"
---

### Body

<ParamField body="api_key" type="string" required>
  This is the API key for your account. You can obtain this from the "Secrets" tab of your "Settings" page.
</ParamField>

<ParamField body="subproject_id" type="string" required>
  This is the id of the subproject the pipeline resides in. You can obtain this from the URL of the subproject page.
</ParamField>

<ParamField body="pipeline_id" type="string" required>
    This is the id of the pipeline whose runs you are trying to query. You can obtain this from the table on the first tab of the pipeline modal.
</ParamField>

### Response

<ResponseField name="status" type="number">
  Indicates whether the call was successful. 200 if successful, 400 if not.
</ResponseField>

<ResponseField name="message" type="[Pipeline Run]">
     Message containing the queried pipeline runs.
    <Expandable title="Pipeline Run Object Structure">
        <ResponseField name="name" type="string" required>
            This is the name of the pipeline run
        </ResponseField>
        <ResponseField name="creator" type="string" optional>
            The name of the user who created the pipeline run
        </ResponseField>
        <ResponseField name="run_date" type="string" optional>
            This is the date the pipeline run was run (MM/DD/YYYY)
        </ResponseField>
        <ResponseField name="id" type="string" optional>
            This is the id of the pipeline run
        </ResponseField>
    </Expandable>
</ResponseField>
