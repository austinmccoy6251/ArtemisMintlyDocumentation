title: "Custom Resource Types"
---
Artemis comes preloaded with some of the most common Resource types enabling you to document scripts, databases, models, etc. 

However, it’s probable that you’ll need to document a file type or other asset that we haven’t preloaded. Fortunately, it’s easy to create a custom type. 

### Walkthrough
This walkthrough demonstrates how to create a custom Resource type.
