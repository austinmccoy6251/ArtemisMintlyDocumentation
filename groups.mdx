---
title: "Creating Groups"
---
Groups are how you create bundles of Instances which you can compare via Views. Groups can contain one or more Instances. Each View can have a practically unlimited number of Groups under it.

Create Groups by visiting a Network, CTRL + Clicking all the Instances you want inside a Group, opening the Views sidebar menu and pressing the green “Add Group” button. This is demonstrated in the walkthroughs below.
