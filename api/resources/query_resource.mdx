---
title: "Query Resource"
api: "GET https://artemisarbackend.com:8080/ext/query_resource"
description: "This endpoint queries and returns a resource given its resource id."
---

### Body

<ParamField body="api_key" type="string" required>
  This is the API key for your account. You can obtain this from the "Secrets" tab of your "Settings" page.
</ParamField>

<ParamField body="subproject_id" type="string" required>
  This is the id of the subproject where the resource is located. You can obtain this ID from the URL while in the subproject.
</ParamField>

<ParamField body="resource_id" type="string" required>
This is the id of the resource to be queried. You can obtain this ID by visiting the resource and looking at the table on the first tab of the resource modal.
</ParamField>

<ParamField body="version" type="string" required>
  This is the version of the resource to be queried. This is indicated in the "Version Dropdown" while viewing the resource. The default version is "Version 1".
</ParamField>

### Response

<ResponseField name="status" type="number">
  Indicates whether the call was successful. 200 if successful, 400 if not.
</ResponseField>

     <ResponseField name="description" type="string">

    </ResponseField>

        The content notes of the resource

    <ResponseField name="content" type="[Content]">
        The content of the resource
         <Expandable title="Content Structure">
            <ResponseField name="name" type="string" required>

            <ResponseField name="version" type="string" optional>
                The version of the content

        </Expandable>
    </ResponseField>
    <ResponseField name="fields" type="[Field]">

    </Expandable>
