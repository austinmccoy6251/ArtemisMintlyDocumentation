---
title: "Query Pipeline Run"
api: "GET https://artemisarbackend.com:8080/ext/query_pipeline_run"
description: "This endpoint queries and returns a specified run of a specified pipeline"
---

### Body

<ParamField body="api_key" type="string" required>
  This is the API key for your account. You can obtain this from the "Secrets" tab of your "Settings" page.
</ParamField>

<ParamField body="subproject_id" type="string" required>
  This is the id of the subproject the pipeline run resides in. You can obtain this from the URL of the subproject page.
</ParamField>

<ParamField body="pipeline_id" type="string" required>
    This is the id of the pipeline that is associated with this run. You can obtain this from the table on the first tab of the pipeline modal.
</ParamField>

<ParamField body="pipeline_version" type="string" required>
    This is the version of the pipeline associated with this run. This is the name of the version indicated in the verison dropdown. The default version is "Version 1".
</ParamField>

<ParamField body="pipeline_run_name" type="string" required>
    This is the name of the pipeline run to be queried.
</ParamField>
